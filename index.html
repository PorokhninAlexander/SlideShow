JavaScript Carousel

Functions:

  • Auto play: next photo every 3 seconds
  • Select: if select photo in photo list, then carousel stoped on this photo
  • Start auto play: if touch on the selected element, them auto play stated
  • Fullscreen: if select photo on carousel, then opened fullscreen window
  • In fullscreen edition:
    • Close: has button in right top angel and 'Escape' on the keyboard
    • Jump to the next photo and to the previous photo: arrow right and left on the keyboard
